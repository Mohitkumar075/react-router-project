React Router Project ->
This is a beginner-friendly React.js project demonstrating routing using React Router.

Features:->
- Single Page Application (SPA) with multiple pages
- Dynamic navigation between components
- Nested routes support
- Clean and simple component structure

Tech Stack:->
- React.js
- React Router v6
- Vite
- JavaScript / JSX / CSS

How to Run:->
1. Clone the repository:
git clone https://github.com/Mohitkumar075/react-router-project.git

2. Go to project folder:
cd react-router-project

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

Open your browser at http://localhost:5173 to see the project in action.
